#### 解题思路：

**先从左往右，从第一列开始，寻找右侧更高的柱子，这两个柱子之间假设以更低柱子的高度全部填充水，再减去两柱子之间的柱子的高度之和即为这两个柱子之间的水的数量。继续往右寻找更高的柱子，直到没有更高的柱子，此时，该柱子就是最高的柱子，从最右边的列开始向左寻找更高的柱子，直到找到这个最高的柱子，思路完全相同，只是要从右往左寻找。**
